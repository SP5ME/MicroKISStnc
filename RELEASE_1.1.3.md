# MicroKISStnc 1.1.3

## Opis aplikacji
MicroKISStnc 1.1.3 to desktopowy TNC APRS/KISS. Aplikacja odbiera ramki KISS z lokalnych klientów APRS, zamienia je na audio AFSK do nadawania oraz dekoduje audio AFSK z powrotem do ramek KISS.

W tej wersji uporządkowano też interfejs webowy tak, aby był spójny z aplikacją desktopową.

## Co jest w tej wersji
- Ujednolicony wygląd Web UI z interfejsem desktopowym.
- Przełącznik języka dostępny także w przeglądarce.
- Możliwość zmiany portu KISS z poziomu Web UI.
- Uproszczony monitor bez zbędnych pól `Audio RX`, `Audio TX`, `PTT Active` i opisu `Live RX / TX stream`.
- Domyślnie uruchamiana karta `Monitor`.
- Ujednolicone nazewnictwo i etykiety w Web UI.
- Zaktualizowany tag wersji aplikacji do `1.1.3`.
- Zaktualizowane nazwy artefaktów builda do `v1.1.3`.

## Gotowy opis do GitHub Release
MicroKISStnc 1.1.3 to desktopowy TNC APRS/KISS z ujednoliconym interfejsem webowym i desktopowym.

Aplikacja:
- odbiera ramki KISS od lokalnych klientów APRS,
- zamienia je na audio AFSK do nadawania,
- dekoduje audio AFSK z powrotem do ramek KISS,
- udostępnia lokalny serwer KISS,
- oferuje Web UI do sterowania i diagnostyki,
- pozwala przełączać język także w przeglądarce,
- pozwala zmieniać port KISS z poziomu Web UI.

Najważniejsze elementy tej wersji:
- ujednolicony Web UI
- przełącznik języka w przeglądarce
- konfiguracja portu KISS z Web UI
- uproszczony monitor
- domyślna karta `Monitor`
