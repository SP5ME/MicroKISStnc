STRESZCZENIE WYKONAWCZE
=======================

TNC-APP (Python middleware) może pracować z Direwolfem (software TNC) na co najmniej 
3 sposoby:

1. **KISS Client Mode** - TNC-APP łączy się do Direwolfa (port 8001)
   Najlepsze dla: Centralnego monitoringu i IGate
   
2. **KISS Server Mode** - Direwolf łączy się do TNC-APP (port 8001)
   Najlepsze dla: Rozproszonego systemu z wieloma TNC-ami
   
3. **Hybrid Mode** - Kombinacja obu poprzednich
   Najlepsze dla: Kompleksowego systemu z redundancją

Obecna architektura TNC-APP jest już przygotowana do obsługi Direwolfa bez zmian.


AKTUALNA ARCHITEKTURA TNC-APP
==============================

Komponenty:
  ✓ KISS Server (TCP 8001) - obsługuje wielorakie połączenia
  ✓ APRS-IS Gateway (TCP 14580) - połączenie z internetem
  ✓ Signal handling - graceful shutdown
  ✓ PacketProcessorThread - przetwarzanie pakietów
  ✓ Logging system - monitorowanie

Przepływ danych (bieżący):
  APRS-IS (Internet) → TNC-APP APRS-IS Gateway → KISSBuilder → 
  Broadcast to KISS clients

Problem: Brakuje obsługi Direwolfa jako źródła pakietów


SCENARIUSZ 1: TNC-APP jako KISS Client dla Direwolfa
======================================================

Architektura:
──────────────

[Radio] → [Direwolf TNC] → [KISS Server 8001] → [TNC-APP KISS Client] 
                                                      ↓
                                              ┌─────────────────┐
                                              ├─ IGate (APRS-IS)│
                                              ├─ KISS Broadcast │
                                              └─────────────────┘

Przepływ danych:
────────────────

1. Direwolf odbiera pakiet z radia
2. Direwolf dekoduje: Radio signal → AX.25 → APRS
3. Direwolf wysyła KISS frame na port 8001
4. TNC-APP KISS Client subprocess łączy się do Direwolfa:8001
5. TNC-APP odbiera KISS frames od Direwolfa
6. TNC-APP przetwarza pakiety (parse APRS, validate)
7. TNC-APP retransmituje do:
   - IGate (APRS-IS internet)
   - Własnych KISS clients (YAAC, Winlink, etc.)
   - Monitoring/logging

```python
class KISSClientToDirectewolf:
    """
    KISS Client connecting to Direwolf (server mode)
    """
    
    def __init__(self, direwolf_host='localhost', direwolf_port=8001):
        self.host = direwolf_host
        self.port = direwolf_port
        self.socket = None
        self.running = False
        self.thread = None
        
    def connect(self):
        """Connect to Direwolf KISS server"""
        try:
            self.socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
            self.socket.settimeout(2.0)
            self.socket.connect((self.host, self.port))
            self.running = True
            logger.info(f"[KISS-CLIENT] Connected to Direwolf {self.host}:{self.port}")
            
            # Start receive thread
            self.thread = threading.Thread(target=self._receive_loop, daemon=False)
            self.thread.start()
        except Exception as e:
            logger.error(f"[KISS-CLIENT] Connection failed: {e}")
            
    def _receive_loop(self):
        """Receive KISS frames from Direwolf"""
        buffer = b''
        FEND = b'\xc0'
        
        while self.running:
            try:
                data = self.socket.recv(1024)
                if not data:
                    break
                    
                buffer += data
                
                # Process KISS frames
                while FEND in buffer:
                    idx = buffer.find(FEND)
                    next_idx = buffer.find(FEND, idx + 1)
                    
                    if next_idx > idx:
                        frame = buffer[idx+1:next_idx]
                        buffer = buffer[next_idx:]
                        self.process_kiss_frame(frame)
                    else:
                        break
                        
            except socket.timeout:
                continue
            except Exception as e:
                logger.error(f"[KISS-CLIENT] Receive error: {e}")
                break
                
    def process_kiss_frame(self, frame):
        """Process received KISS frame from Direwolf"""
        if len(frame) < 2:
            return
            
        cmd = frame[0]
        
        if cmd == 0x00:  # Data frame
            # Unescape frame
            unescaped = self.unescape_kiss(frame[1:])
            
            # Process as AX.25 frame
            packet_data = {
                'source': 'direwolf',
                'kiss_frame': unescaped,
                'timestamp': datetime.now().isoformat()
            }
            
            # Send to packet processor
            logger.info(f"[KISS-CLIENT] Received frame from Direwolf: {len(unescaped)} bytes")
            # emit_packet(packet_data)
            
    @staticmethod
    def unescape_kiss(data):
        """Unescape KISS frame"""
        result = b''
        i = 0
        while i < len(data):
            if data[i:i+1] == b'\xdb':
                if i+1 < len(data):
                    if data[i+1:i+2] == b'\xdc':
                        result += b'\xc0'
                        i += 2
                    elif data[i+1:i+2] == b'\xdd':
                        result += b'\xdb'
                        i += 2
                    else:
                        result += data[i:i+1]
                        i += 1
                else:
                    result += data[i:i+1]
                    i += 1
            else:
                result += data[i:i+1]
                i += 1
        return result
        
    def stop(self):
        """Stop KISS client"""
        self.running = False
        if self.socket:
            try:
                self.socket.close()
            except:
                pass
        if self.thread:
            self.thread.join(timeout=2.0)
```

**Problem**: Direwolf NIE obsługuje KISS client mode (tylko server mode)!

Direwolf expects applications to connect TO it, not the other way around.

Rozwiązanie: Użyć adaptera pośredniego lub zmodyfikować Direwolfa (kompilacja z zmianami).

Wada tego scenariusza:
──────────────────────
  ✗ Wymaga modyfikacji Direwolfa
  ✗ Wymaga rekomilacji
  ✗ Brak wsparcia w oficjalnej wersji
  ✗ Maintenance nightmare

```python
"""
KISS Client module - Connects to Direwolf and other KISS servers
"""

import socket
import threading
import logging
from datetime import datetime
from typing import Callable, Optional

logger = logging.getLogger(__name__)

class KISSClient:
    def __init__(self, host: str = 'localhost', port: int = 8001):
        self.host = host
        self.port = port
        self.socket: Optional[socket.socket] = None
        self.running = False
        self.thread: Optional[threading.Thread] = None
        self.on_frame_received: Optional[Callable] = None
        
    def set_frame_callback(self, callback: Callable):
        """Set callback for received KISS frames"""
        self.on_frame_received = callback
        
    def connect(self) -> bool:
        """Connect to KISS server"""
        try:
            self.socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
            self.socket.settimeout(2.0)
            self.socket.connect((self.host, self.port))
            self.running = True
            
            logger.info(f"[KISS-CLIENT] Connected to {self.host}:{self.port}")
            
            # Start receive thread
            self.thread = threading.Thread(
                target=self._receive_loop,
                name=f"KISSClient-{self.host}:{self.port}",
                daemon=False
            )
            self.thread.start()
            return True
            
        except Exception as e:
            logger.error(f"[KISS-CLIENT] Connection failed to {self.host}:{self.port}: {e}")
            return False
            
    def _receive_loop(self):
        """Receive KISS frames"""
        buffer = b''
        FEND = 0xC0
        
        while self.running and self.socket:
            try:
                data = self.socket.recv(1024)
                if not data:
                    logger.info("[KISS-CLIENT] Connection closed by server")
                    break
                    
                buffer += data
                
                # Process complete KISS frames
                while FEND in buffer or buffer.startswith(bytes([FEND])):
                    if not buffer.startswith(bytes([FEND])):
                        # Skip to next FEND
                        idx = buffer.find(FEND)
                        buffer = buffer[idx:]
                        continue
                        
                    # Find frame end
                    end_idx = buffer.find(FEND, 1)
                    if end_idx == -1:
                        break  # Incomplete frame
                        
                    # Extract frame
                    frame = buffer[1:end_idx]
                    buffer = buffer[end_idx:]
                    
                    # Process frame
                    self._process_kiss_frame(frame)
                    
            except socket.timeout:
                continue
            except Exception as e:
                logger.error(f"[KISS-CLIENT] Receive error: {e}")
                break
                
        self.running = False
        if self.socket:
            try:
                self.socket.close()
            except:
                pass
                
    def _process_kiss_frame(self, frame: bytes):
        """Process single KISS frame"""
        if not frame:
            return
            
        cmd = frame[0]
        
        if cmd == 0x00:  # Data frame
            # Unescape
            unescaped = self._unescape_kiss(frame[1:])
            
            # Callback
            if self.on_frame_received:
                try:
                    self.on_frame_received({
                        'source': f"{self.host}:{self.port}",
                        'data': unescaped,
                        'timestamp': datetime.now().isoformat(),
                        'length': len(unescaped)
                    })
                except Exception as e:
                    logger.error(f"[KISS-CLIENT] Callback error: {e}")
        else:
            logger.debug(f"[KISS-CLIENT] Command frame: {cmd:02x}")
            
    @staticmethod
    def _unescape_kiss(data: bytes) -> bytes:
        """Unescape KISS data"""
        result = b''
        i = 0
        while i < len(data):
            if data[i] == 0xDB and i + 1 < len(data):
                if data[i + 1] == 0xDC:
                    result += bytes([0xC0])
                    i += 2
                elif data[i + 1] == 0xDD:
                    result += bytes([0xDB])
                    i += 2
                else:
                    result += bytes([data[i]])
                    i += 1
            else:
                result += bytes([data[i]])
                i += 1
        return result
        
    def disconnect(self):
        """Disconnect from KISS server"""
        self.running = False
        if self.socket:
            try:
                self.socket.close()
            except:
                pass
        if self.thread:
            self.thread.join(timeout=2.0)
        logger.info(f"[KISS-CLIENT] Disconnected from {self.host}:{self.port}")
        
    def is_connected(self) -> bool:
        """Check if connected"""
        return self.running and self.socket is not None
```
